MAC 环境下构建 PHP 自动化环境工具 V2.0

Posted on 2021-10-05

前段时间手撸一个 PHP 的本地自动化环境脚本, 跟朋友分享了一波之后收到了绝大多数的好评. 但是在实际使用的过程中还是有不少的问题, 因为刚入职了新公司也没有去升级, 今天刚好有空抽了两个小时升级了一下.

实际使用中遇到的问题:

  • 依赖 Mac 自带的 Apache: 本来这个工具就是因为我懒得每次有个啥新项目都需要去重新部署本地环境, 虽然十分钟就能搞定, 但还是懒得搞, 所以本身就是为我自己本机环境定制的. 新公司配的电脑是 Linux, 诶!, 难受了, 就需要移植到 Linux 上
  • 域名 和 Nginx 中的 root 固定不可配: 本来想是为 Laravel 和 Lumen 定制的工具, 但是实际项目中有 Yii 的项目, 且一个项目可能有多个入口, 每次都只能手动去修改, 很不方便

就上述两个问题, 对脚本进行本次升级. 主要思想上次也提到过, 就是通过 docker 拉起一个 nginx 的代理镜像, 通过代理镜像把流量转发到各个业务镜像上

项目地址: https://gogs.ijunj.com/common/php-link-tool

工具在 mac 和 ubuntu 系统下, 运行正常, 要不是我实在搞不懂 windows, 我真想也搞上, 这样才是一个完整的脚本嘛
